# 中国民商事诉请/抗辩要件模板

本文件为 `claim-chart` 的基础要件库。具体案件以现行法律、司法解释、指导案例/典型案例、地方裁审口径和用户提供材料为准。

## 使用前治理要求

- 仅将本模板用作非权威的问题识别辅助，不将其视为现行法的完整陈述。
- 使用前记录法域、依据层级、施行/失效及核验日期、来源和版本日期，并由具备相应资格的人员核验。
- 根据具体案件补充不利事实、例外、抗辩、证明责任/证明标准和救济条件。
- 不自动选择请求权基础，不自动认定要件满足；控制性依据缺失、过期或适用性不明时，标记为待核验并阻止后续实质使用。
- 保留不确定性，并记录具体分析与模板之间的偏离。

## 买卖合同价款请求

| 要件 | 常见证据 |
|---|---|
| 合同成立并生效 | 合同、订单、报价、盖章/电子签、往来邮件 |
| 卖方已交付或完成主要义务 | 送货单、签收单、验收单、物流、系统记录 |
| 买方付款条件已成就 | 验收、发票、对账单、付款节点 |
| 买方逾期未付款 | 付款流水、催款函、微信/邮件 |
| 金额和违约责任 | 对账单、发票、利息/违约金计算 |

## 服务合同报酬请求

| 要件 | 常见证据 |
|---|---|
| 服务合同关系 | 合同、SOW、订单 |
| 服务已履行 | 交付物、项目记录、会议纪要、验收、使用记录 |
| 付款条件成就 | 里程碑、验收、对账、发票 |
| 对方未付款或拖延验收 | 催告记录、对方回复 |
| 损失或违约金 | 计算表、合同条款 |

## 解除合同/继续履行抗辩

| 要件 | 常见证据 |
|---|---|
| 对方根本违约或约定解除条件成就 | 合同条款、违约事实、催告 |
| 解除通知已有效送达 | EMS、电子邮件、合同约定系统、签收记录 |
| 我方无先违约或先履行抗辩成立 | 履行记录、对方违约证据 |
| 损失和返还范围 | 付款、库存、费用、折旧、替代交易 |

## 侵权损害赔偿

| 要件 | 常见证据 |
|---|---|
| 侵权行为 | 照片、视频、平台页面、公证、鉴定 |
| 损害结果 | 维修、损失、销售下降、医疗/财产证明 |
| 因果关系 | 时间线、专家意见、现场记录 |
| 过错或归责基础 | 警示、管理记录、技术标准、监管认定 |
| 赔偿金额 | 票据、评估、鉴定、合理支出 |

## 公司决议纠纷

| 要件 | 常见证据 |
|---|---|
| 股东/董事主体资格 | 工商档案、股东名册、章程 |
| 会议召集和通知 | 通知、送达、会议材料 |
| 表决程序和比例 | 会议记录、表决票、签字 |
| 决议内容 | 决议文本 |
| 撤销/无效/不成立事由 | 公司法、章程、程序瑕疵证据 |

## 执行异议

| 要件 | 常见证据 |
|---|---|
| 执行依据和执行行为 | 判决/裁决、执行裁定、查封冻结文书 |
| 异议主体资格 | 权属证明、利害关系 |
| 排除执行或变更执行理由 | 买卖合同、付款、占有、登记、唯一住房等证据 |
| 救济期限 | 收到执行文书日期、提出异议日期 |

## 通用抗辩

- 诉讼时效/仲裁时效。
- 管辖/仲裁协议异议。
- 主体不适格。
- 合同未成立、无效、可撤销或未生效。
- 同时履行抗辩、先履行抗辩、不安抗辩。
- 已履行、抵销、免除、混同、债务转移。
- 不可抗力、情势变更。
- 违约金过高调整。
