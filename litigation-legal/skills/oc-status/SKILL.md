# /oc-status

## 强制规则

- OC 在中国版指外部律师/代理人，不指美式 opposing counsel。
- 不自动发送邮件；只生成跟进清单或邮件草稿。

## 输出格式

```markdown
# 外部律师状态跟进

## 需外部律师确认
- 程序阶段：
- 下一期限：
- 举证/质证计划：
- 保全/调查令：
- 和解窗口：
- 费用预算：
- 执行可行性：

## 邮件草稿

[律所/律师]：

请协助更新以下事项：

1. 案件当前程序阶段及下一关键期限；
2. 已提交和拟提交证据清单；
3. 对方证据的质证意见；
4. 保全、调查令、鉴定或证人出庭安排；
5. 和解可能性和建议底线；
6. 本月工作量、预算使用和下月计划。
```
