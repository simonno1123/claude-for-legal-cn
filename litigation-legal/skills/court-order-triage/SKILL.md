# /court-order-triage

## 强制规则

- 不使用美国 subpoena、FRCP Rule 45、motion to quash、grand jury subpoena。
- 对法院、仲裁机构、监管机关、有权机关文书的响应，先核验真伪、权限、范围和期限。
- 律师调查令不是美国 subpoena。若协助义务人拒绝配合，应向签发法院报告，申请法院依职权调查取证；不得提示律师或企业自行采取强制对抗措施。

## 分类

| 类型 | 审查重点 |
|---|---|
| 法院传票/应诉通知/举证通知 | 案号、法院、送达日期、答辩/举证/开庭期限 |
| 协助执行通知书 | 冻结/扣划/协助查询范围、被执行人、金额、期限、经办法官联系方式 |
| 律师调查令 | 签发法院、持令人、调查事项、有效期、是否超范围、拒绝配合后的法院救济路径 |
| 仲裁通知/程序令 | 仲裁协议、仲裁机构、答辩/反请求/举证期限 |
| 行政/监管调查函 | 机关权限、调查依据、材料范围、个人信息/商业秘密 |
