# /subpoena-triage

> 兼容命令：本命令已由 `/litigation-legal:court-order-triage` 取代。收到法院文书、协助执行通知、律师调查令或监管函时，优先使用新命令；本文件仅保留路由兼容。

## 强制规则

- 本技能在中国法语境下等同 `court-order-triage`。
- 不使用 FRCP Rule 45、motion to quash、deposition subpoena、grand jury subpoena、privilege log。
- 识别为律师调查令时，必须提示：若协助义务人拒绝配合，应向签发法院报告并申请法院依职权调查取证，不得提示自行采取强制措施。
- 调用 `references/china-litigation-core-rules.md`。
- 企业对法院、仲裁机构、有权机关文书的响应必须先核验真伪、权限、范围和期限。

## 分类

| 类型 | 审查重点 |
|---|---|
| 法院传票/应诉通知/举证通知 | 案号、法院、送达日期、答辩/举证/开庭期限 |
| 协助执行通知书 | 冻结/扣划/协助查询范围、被执行人、金额、期限、经办法官联系方式 |
| 律师调查令 | 签发法院、持令人、调查事项、有效期、是否超范围 |
| 行政/监管调查函 | 机关权限、调查依据、材料范围、个人信息/商业秘密 |
| 仲裁通知/程序令 | 仲裁协议、仲裁机构、答辩/反请求/举证期限 |

## 工作流

1. **真伪核验**
   - 核对法院/机关名称、案号、公章、承办法官/书记员、联系电话、官方渠道。
   - 扫描件、截图、微信转发文书不得直接响应，应通过官方电话/平台复核。

2. **权限和范围**
   - 文书要求是否由有权机关出具。
   - 要求提供的信息、冻结/扣划的资产、调查的事项是否超出文书载明范围。

3. **期限**
   - 提取送达日期、答复日期、举证期限、开庭日期、协助履行期限。
   - 标注节假日和顺延因素 `[待律师复核]`。

4. **个人信息/商业秘密**
   - 员工信息、客户数据、交易记录、源代码、财务资料只按最小必要范围提供。
   - 涉商业秘密时提示申请不公开、限制阅卷、密封提交或脱敏。

5. **响应路径**
   - 需要配合、申请延期、书面说明、范围异议、复议/执行异议、律师复核。
