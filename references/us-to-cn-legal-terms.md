# 美国法术语到中国法语境替换表

在中国大陆法域下，模型必须丢弃美国普通法、美国证据开示制度和美国律师特权框架，不得把下列术语直译为中国法中不存在或效力不同的制度。

| 原术语 | 中国法语境替换 | 强制说明 |
|---|---|---|
| `attorney work product` / `work product doctrine` | 律师工作成果初稿 / 内部法律分析 / 保密文件 | 不得承诺可对抗法院、仲裁机构或监管机关的强制调查、取证或披露要求；仅说明保密管理和审阅属性。 |
| `attorney-client privilege` | 律师保密义务 / 委托关系保密 / 律师执业保密义务 | 不得承诺美国式证据特权；中国语境下应表述为保密义务、内部控制和披露风险。 |
| `discovery` | 举证、证据交换、调查取证、法院调查令、责令提交书证 | 按具体场景区分，不得默认存在美国式广泛证据开示程序。 |
| `deposition` | 询问笔录、庭审发问提纲、证人证言、调查谈话记录 | 中国民事诉讼中通常不使用美国式庭外宣誓证言制度。 |
| `subpoena` | 法院传票 / 调查令 / 协助调查通知 / 责令提交材料 | 必须按主体和场景区分：法院、仲裁机构、监管机关、律师调查令等不能混用。 |
| `privilege log` | 敏感材料审查表 / 不提交理由清单 / 保密材料清单 | 不得承诺“特权”保护；仅可作为内部审查和向律师说明的材料管理表。 |
| `DMCA takedown` | 平台知识产权投诉 / 通知删除 / 反通知 / 电商平台知识产权保护规则 | 中国语境下优先适用《民法典》《电子商务法》《著作权法》《商标法》《专利法》及平台规则。 |
| `DSAR` | 个人信息主体权利请求 | 对应《个保法》下查阅、复制、更正、补充、删除、撤回同意、解释说明、转移等个人权利请求。 |
| `state bar` | 律师协会 / 司法行政机关 / 12348 公共法律服务热线 | 不得引导用户联系美国州律师协会；中国用户应联系当地律师协会、司法行政机关或 12348。 |
| `licensed attorney` | 中国执业律师 / 律师事务所 / 企业法务 | 按用户身份区分执业律师、企业法务、非律师业务人员。 |
| `governing law: Delaware/New York/California` | 适用中华人民共和国大陆地区法律；涉外合同另行确认准据法 | 第一阶段不得默认推荐美国州法。 |
| `FLSA / FMLA / ADA / WARN` | 《劳动法》《劳动合同法》《社会保险法》《劳动争议调解仲裁法》及地方工时休假/裁员规则 | 美国劳动法缩写仅在跨境或境外雇佣场景作为比较法信息出现。 |
| `GDPR / CCPA` | 《个保法》《数安法》《网安法》及数据出境配套规则 | 中国版默认不以 GDPR/CCPA 为主框架；跨境业务可作为并行比较。 |
| `USPTO` | 国家知识产权局 / 商标局 / 中国专利检索及分析系统 | 中国知识产权工作流默认使用 CNIPA 及中国平台/法院/行政机关路径。 |

## 输出强制规则

- 不得使用“证据特权”“开示豁免”“法院不能要求披露”等会造成错误安全感的表述。
- 如原材料中出现美国法术语，必须先判断是否为跨境/境外法场景；若不是，按本表转换为中国法语境。
- 如用户明确要求分析美国法，应提示该问题超出中国版默认法域，并建议由相应法域律师确认。
