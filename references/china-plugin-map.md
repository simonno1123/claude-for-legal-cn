# 中国法律模块映射

| 插件 | 中国化定位 | 核心法律/制度 |
|---|---|---|
| `commercial-legal` | 商事合同审查、续约、授权审批、争议解决条款 | 《民法典》合同编、《仲裁法》《民诉法》《电子签名法》、发票税务与印章授权规则 |
| `privacy-legal` | 个人信息保护、数据合规、数据出境、个人信息权利请求 | 《个保法》《数安法》《网安法》及网信部门配套规则 |
| `product-legal` | 产品发布、营销宣传、消费者权益、平台治理 | 《广告法》《消保法》《电商法》《反不正当竞争法》、网信与市场监管规则 |
| `corporate-legal` | 公司治理、投融资、并购尽调、交割、工商合规 | 《公司法》《证券法》《外商投资法》、国资与市场监管规则 |
| `employment-legal` | 招聘、劳动合同、解除终止、工时休假、员工手册、劳动争议 | 《劳动法》《劳动合同法》《社保法》《劳动争议调解仲裁法》及地方人社口径 |
| `regulatory-legal` | 法规政策监测、差距分析、制度修订、意见征集 | 法律法规、行政法规、部门规章、司法解释、规范性文件、征求意见稿 |
| `ai-governance-legal` | 企业 AI 使用、生成式 AI、算法推荐、深度合成、模型/服务治理 | 生成式 AI、算法推荐、深度合成、数据安全、个人信息保护、科技伦理规则 |
| `ip-legal` | 商标、专利、著作权、商业秘密、开源与侵权初筛 | 《商标法》《专利法》《著作权法》《反不正当竞争法》、CNIPA/版权登记/平台规则 |
| `litigation-legal` | 民商事诉讼、仲裁、证据、保全、案件管理、外部律师协作 | 《民诉法》《仲裁法》、证据规则、保全、管辖、执行、劳动仲裁前置 |
| `legal-clinic` | 法律诊所、法律援助、学生接待、导师审阅、案件备忘 | 法律援助制度、12348、律师执业边界、教育/诊所管理规则 |
| `law-student` | 中国法学习、法考、案例研读、请求权基础、法条训练 | 中国部门法体系、司法解释、指导案例/典型案例、法考大纲 |
| `legal-builder-hub` | 中国法律技能的安装、审查、更新和禁用 | 技能安全、资料源可信度、MCP 权限、提示词注入防护 |
