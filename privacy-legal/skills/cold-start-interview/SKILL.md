# /cold-start-interview

## 强制规则

- 默认适用中国大陆《个人信息保护法》《数据安全法》《网络安全法》。
- 不得询问 GDPR/CCPA/HIPAA/DSAR/DPIA 作为默认配置；涉外时才作为扩展问题。
- 输出 `privacy_profile`，作为本插件唯一初始化底座；`customize` 只允许局部更新。

## 访谈维度

1. **主体与行业**
   - 企业名称、注册地、主要经营地、行业、监管机关、是否涉金融、医疗、教育、未成年人、交通、政务或平台经营。
   - 是否可能属于关键信息基础设施运营者、是否处理重要数据 `[待查证]`。

2. **产品与系统**
   - App、小程序、网站、SaaS、IoT、线下门店、HR、CRM、客服、营销、风控、AI 系统。
   - 主要用户：消费者、企业客户、员工、未成年人、供应商、商户。

3. **个人信息与敏感信息**
   - 收集的信息类型、来源、目的、频率、保存期限和删除规则。
   - 生物识别、位置轨迹、医疗健康、金融账户、身份证件、未成年人信息等敏感个人信息。

4. **同意和告知**
   - 隐私政策、弹窗、权限申请、单独同意、撤回同意、注销账号。
   - 是否存在一揽子同意、默认勾选、拒绝授权后无法使用非必要功能。

5. **第三方和跨境**
   - SDK、云服务、支付、物流、客服、广告、数据分析、集团共享、外包供应商。
   - 境外接收方、国家/地区、数据类型、规模、传输目的和出境路径。

6. **治理流程**
   - PIPIA 模板和留痕、个人权利请求、数据安全事件响应、供应商审计、日志留存、员工权限控制。
