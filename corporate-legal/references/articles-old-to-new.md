# 公司章程旧法残留到新公司法口径替换表

本文件是 `corporate-legal` 章程审计、实体合规、公司治理审计和股权变动审查的底层规则字典。用于识别旧公司法、旧外资三法、美国公司治理模板或历史章程中的高风险条款，并给出新公司法语境下的修订方向。

> 注意：本表仅提供 Prompt 对齐规则。具体条文序号、过渡期、市场监管备案口径、外资/国资/上市公司特别规则必须结合国家法律法规数据库、市场监管部门和律师复核。

## 1. 股权转让审批程序

| 项目 | 内容 |
|---|---|
| 旧法残留条款 | “公司股东向股东以外的人转让股权，应当经其他股东过半数同意。” |
| 风险 | 该表述沿用旧公司法默认审批逻辑，可能导致企业错误追求“同意票”，忽视新公司法下书面通知和优先购买权流程。 |
| 新公司法规范口径 | 股东向股东以外的人转让股权的，应当将转让数量、价格、支付方式和期限等事项书面通知其他股东。其他股东自收到书面通知之日起三十日内未答复的，视为放弃优先购买权。 |
| Prompt 卡点 | 强制审查书面通知内容、送达留痕、三十日回复、优先购买权放弃/行使证据和章程特别约定。 |
| 参考条文 | 《公司法》第84条 `[待查证]` |

## 2. 公司权力机构定位

| 项目 | 内容 |
|---|---|
| 旧法残留/外资过渡条款 | “董事会是公司的最高权力机构”；或“股东会是公司的最高权力机构”。 |
| 风险 | 老中外合资企业常以董事会作为最高权力机构，外资三法转轨后可能与公司法治理结构冲突；“最高权力机构”表述也不宜继续机械沿用。 |
| 新公司法规范口径 | 公司股东会由全体股东组成，是公司的权力机构。章程可在法律框架内细化股东会、董事会/董事、经理层权限。 |
| Prompt 卡点 | 强制识别公司类型、外资/国资/上市标签、章程是否已在 2024 年后修订、股东会和董事会权限是否混淆。 |
| 参考条文 | 《公司法》第58条 `[待查证]` |

## 3. 监督机制：监事会、监事与审计委员会

| 项目 | 内容 |
|---|---|
| 旧法残留条款 | “公司设监事会/设监事一名”且未提供审计委员会路径；或仅套用美国式 Audit Committee。 |
| 风险 | 新公司法下公司可根据章程选择监事会/监事或董事会审计委员会路径，不能机械保留旧监督机关，也不能直接套用美国审计委员会概念。 |
| 新公司法规范口径 | 公司可按章程规定在董事会中设置由董事组成的审计委员会，行使监事会职权，不设监事会或监事。小规模或股东人数较少的公司，在特定条件下也可不设监事。 |
| Prompt 卡点 | 审查公司类型、董事会设置、审计委员会成员和职责、职工董事/职工监事、章程依据、备案材料。 |
| 参考条文 | 《公司法》第69条、第83条 `[待查证]` |

## 4. 出资期限硬性约束

| 项目 | 内容 |
|---|---|
| 旧法残留条款 | “股东认缴的出资应当在公司设立之日起二十年内缴足”；“股东可按章程约定期限缴足”；“不设出资期限”。 |
| 风险 | 新设有限责任公司若出资期限超过五年，存在直接不合规风险；存量公司需关注过渡整改。 |
| 新公司法规范口径 | 全体股东认缴的出资额由股东按照公司章程规定自公司成立之日起五年内缴足；存量公司过渡安排需另行查证。 |
| Prompt 卡点 | 强制审查成立日期、章程出资期限、各股东认缴/实缴、非货币财产出资、是否需章程整改。 |
| 参考条文 | 《公司法》第47条 `[待查证]` |

## 5. 法定代表人产生与权责

| 项目 | 内容 |
|---|---|
| 旧法残留条款 | “董事长/执行董事或经理担任法定代表人”；未说明执行公司事务、辞任、越权限制和登记变更。 |
| 风险 | 法定代表人权限、辞任、越权签署、印章控制和登记公示不清，会引发合同效力、内部追责和登记风险。 |
| 新公司法规范口径 | 法定代表人由代表公司执行公司事务的董事或者经理担任；章程应明确产生方式、辞任衔接、职权限制和变更登记。 |
| Prompt 卡点 | 审查法定代表人是否实际执行公司事务、章程授权边界、印章/证照/电子签章控制、越权签署和辞任后的变更登记。 |
| 参考条文 | 《公司法》第10条 `[待查证]` |

## 6. 董监高忠实勤勉义务与影子董事

| 项目 | 内容 |
|---|---|
| 旧法残留条款 | “董事、监事、高级管理人员对公司负有忠实义务和勤勉义务。”仅作原则性表述。 |
| 风险 | 未细化关联交易、竞业、商业机会、资金占用、控制股东/实控人指令等场景，难以支撑内部追责和治理审计。 |
| 新公司法规范口径 | 章程和治理制度应细化忠实义务、勤勉义务、关联交易、竞业禁止、商业机会、信息披露、赔偿责任和控制股东/实际控制人指使责任。 |
| Prompt 卡点 | 强制审查董监高责任条款、关联交易审批、商业机会规则、竞业规则、影子董事/事实董事、控制股东和实际控制人连带责任。 |
| 参考条文 | 《公司法》第180-184条、第192条 `[待查证]` |

## 7. 出资催缴与董事责任

| 项目 | 内容 |
|---|---|
| 旧法残留条款 | 章程只写“股东按期缴纳出资”，未规定董事会核查和催缴机制。 |
| 风险 | 新公司法强化董事对股东出资的核查和催缴义务；怠于催缴可能引发董事赔偿责任。 |
| 新公司法规范口径 | 董事会/董事应核查股东出资情况，发现未按期足额出资的，应向股东发出书面催缴通知；未履行义务给公司造成损失的，负有责任的董事可能承担赔偿责任。 |
| Prompt 卡点 | 强制审查出资台账、催缴通知、宽限期、失权程序、董事会记录和董事责任。 |
| 参考条文 | 《公司法》第51条、第52条 `[待查证]` |

## 8. 市场监管年报与出资公示

| 项目 | 内容 |
|---|---|
| 旧法残留条款 | 年报只视作行政填报，未与章程、出资表、实缴凭证核对。 |
| 风险 | 年报公示的股东认缴/实缴、出资方式、出资期限与内部材料不一致，可能引发行政处罚、经营异常和交易尽调风险。 |
| 新公司法规范口径 | 年报和公示信息应真实、准确、完整；股东认缴和实缴信息应与章程、股东名册、出资凭证一致。 |
| Prompt 卡点 | 强制对比国家企业信用信息公示系统、章程、股东名册、出资凭证、市场监管变更/备案材料。 |
| 参考来源 | 国家企业信用信息公示系统、市场监管规则 `[市场监管口径 - 待查证]` |

## 输出要求

章程审计或实体合规技能发现上述旧法残留时，应输出：

1. 旧条款原文或摘要。
2. 风险等级。
3. 新公司法替换口径。
4. 需要补充的章程、决议、登记/备案、股东通知或出资材料。
5. 是否需要律师、市场监管窗口、国资/外资/上市专项复核。
