# 中国常见期限合理性提示

本文件仅用于发现明显异常的期限录入，不自动计算最终期限。具体期限必须按案由、程序、送达方式、当地规则和主管机关要求复核。

| 事项 | 常见范围 | 提醒 |
|---|---|---|
| 普通民事诉讼时效 | 通常为3年 | 起算、中止、中断、最长保护期需核查 |
| 劳动仲裁申请 | 通常为1年 | 劳动关系存续期间工资争议有特殊口径 |
| 行政复议申请 | 通常为60日 | 法律另有规定除外 |
| 行政诉讼起诉 | 通常为6个月 | 复议后起诉期限另行核查 |
| 民事上诉 | 判决通常15日，裁定通常10日 | 从送达次日起算，需核对文书 |
| 举证期限 | 以法院/仲裁委通知为准 | 不得凭经验自动确定 |
| 消费投诉 | 依平台、消协或监管机关规则 | 注意七日无理由退货和质保期限 |
