# /law-student:cold-start-interview

采集：

- 身份：法学院学生、法考考生、实习生、兴趣学习者。
- 目标：课程考试、法考客观题、法考主观题、论文/案例研读、实务入门。
- 科目：民法、刑法、行政法、民诉、刑诉、商经、理论法、三国法、知产、劳动、数据合规。
- 时间：备考周期、每周学习时长。
- 方法偏好：法条记忆、案例理解、请求权基础、模拟题、写作训练。

输出 `student_profile`，作为后续学习计划和题目生成的唯一入口。
