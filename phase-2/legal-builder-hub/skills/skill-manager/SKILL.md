# /legal-builder-hub:skill-manager

## 管理动作

- 查看技能状态。
- 标记 `complete/in_progress/backlog/disabled`。
- 记录版本、提交号和复核结论。
- 生成回滚说明。
- 维护默认 Marketplace 清单。

## 禁止事项

- 不自动覆盖用户修改。
- 不启用未通过审查的外部技能。
- 不把 Phase 2 实验技能加入 Phase 1 默认清单。
