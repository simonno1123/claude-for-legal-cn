# Legal Builder Hub (中国法律技能构建中心)

本模块用于 Phase 2 的技能生态建设：发现、审查、安装、禁用、更新和质检中国法本土化技能。

## 核心能力

- 中国法律技能质量审查
- MCP 连接器占位配置检查
- 插件/技能安装与禁用流程
- 回归测试与乱码/JSON/frontmatter 验证
- Gemini/人工复核包生成的流程约束

## 推荐命令

- `/legal-builder-hub:cold-start-interview`
- `/legal-builder-hub:registry-browser`
- `/legal-builder-hub:skills-qa`
- `/legal-builder-hub:skill-installer`
- `/legal-builder-hub:skill-manager`

## 不做的事

- 不提供实体法律意见。
- 不绕过人工复核安装未审查技能。
- 不保存真实 API Key 或商业数据库凭证。
