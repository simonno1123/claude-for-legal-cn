# /is-this-a-problem

## 适用场景

用于快速判断，不替代完整上线审查。若问题涉及未成年人、敏感个人信息、自动化决策、价格促销、特殊广告、实物质量安全、直播带货或监管投诉，默认升级。

## 快速分流

| Call | 含义 |
|---|---|
| Fine | 低风险，可继续推进，但不等于正式批准 |
| Needs Review | 需要运行 `launch-review`、`marketing-claims-review`、`privacy-legal` 或 `ai-governance-legal` |
| Hold | 暂停对外承诺/发布/上线，先由法务或律师复核 |

## Trap checks

只问一个最关键问题，再给分流结论：

| 问题类型 | 关键追问 |
|---|---|
| 使用客户 logo/案例 | 是否有书面授权？是否暗示背书？ |
| 说“第一/唯一/最” | 有无权威、现行、可公开证明依据？ |
| 自动续费/会员 | 是否显著提示、单独同意、可便捷取消？ |
| A/B 测试价格 | 是否同等条件、是否可能构成价格欺诈或差别待遇？ |
| 训练/推荐/画像 | 用户是否已被告知处理目的？是否涉及敏感/儿童信息？ |
| 默认勾选/搭售 | 是否损害消费者自主选择权？ |
| 直播/达人种草 | 是否显著标识广告？素材和商品资质是否可核验？ |
| 实物/硬件上线 | 是否涉及强制标准、CCC、警示说明、召回风险？ |

## Output

