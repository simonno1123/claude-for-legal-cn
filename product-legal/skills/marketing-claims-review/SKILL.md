# /marketing-claims-review

## 适用场景

当用户问“这句广告能不能说”“宣传页有没有广告法风险”“直播脚本/达人种草/比较宣传/价格促销能否发布”时，使用本技能。

## 强制边界

本技能按中国《广告法》《互联网广告管理办法》《反不正当竞争法》《消费者权益保护法》《价格法》和市场监管口径输出。不得用境外广告审查中的“夸张性宣传可免责”逻辑直接放行。

## 中国广告分类

逐条抽取文案中的宣传性表述，并分类：

| 类别 | 风险规则 |
|---|---|
| 绝对化用语 | “国家级、最高级、最佳、第一、唯一、顶级”等默认高风险，需审查是否属于法律允许的例外 |
| 事实性功效/性能声明 | 必须有可核验证据，如检测报告、统计口径、样本、时间范围 |
| 比较宣传 | 不得虚假、片面或贬损竞争对手；需有同条件、可复核的比较依据 |
| 虚假或引人误解宣传 | 包括夸大功能、遗漏限制条件、暗示不存在的资质或效果 |
| 价格促销 | 明码标价、划线价、到手价、限时优惠、满减、自动续费、默认勾选 |
| 用户评价/案例/代言 | 真实使用、显著标识广告，不得虚构交易或评价，代言需符合身份和使用事实 |
| 直播带货/达人种草 | 广告标识、主播/平台/商家责任、商品资质、退换货和售后 |
| 特殊广告 | 医疗、药品、医疗器械、保健食品、特殊医学用途配方食品、金融、教育培训、招商投资、房地产等 |
| 未成年人相关 | 不得诱导消费，不得损害身心健康，儿童个人信息和内容安全需专项检查 |

## Review workflow

1. 抽取所有具体宣传性表述，不只看用户标出的句子。
2. 判断是否属于广告、商业宣传、平台规则说明、客服承诺或价格促销。
3. 按中国广告分类逐条标注风险。
4. 对每条非低风险表述给出可直接替换的中文改写。
5. 涉及特殊行业、直播带货、价格促销、未成年人或个人信息时，必须标注专项审查。

| 声明 | 所需依据 | 责任人 |

## 禁止输出

- 不得以“夸张性宣传”直接放行。
- 不得在没有依据时认可“第一、唯一、最、100%、保证、永久、国家级”等表述。
- 不得用境外广告审查标准替代中国广告法和市场监管口径。
- 不得输出正式发布批准；只能输出审查初稿和需复核事项。
