# 商事合同法务插件（中国大陆版）

面向中国大陆企业法务、合同管理、采购、销售、信息安全和外部律师的 B2B 商事合同审查与履约管理工作流。默认适用中华人民共和国大陆地区法律，重点覆盖采购合同、销售合同、SaaS/云服务合同、保密协议、框架协议、订单、补充协议、续约提醒和审批流转。

> **专业提示：** 本插件输出均为内部法律审阅初稿。涉及签署、盖章、付款、解除、续约、对外发送、仲裁/诉讼、监管申报或放弃权利前，必须由中国执业律师或企业法务复核确认。

## 适用场景

- 采购/供应商合同审查。
- 销售侧框架协议、订单、服务协议、补充协议审查。
- SaaS、软件许可、云服务、技术服务和数据处理条款审查。
- NDA/保密协议/商业秘密保护协议快速分流。
- 自动续约、到期、取消通知和涨价提醒。
- 合同审批、用印、发票税务、信息安全和数据合规升级。
- 补充协议、变更单和历史版本的现行条款梳理。

## 不适用场景

- 劳动用工、公司治理、诉讼仲裁代理、专利/商标申请、金融证券发行。
- 默认不处理港澳台或境外法律结论；如合同含涉外因素，仅提示需当地律师或专项复核。
- 不自动签署、盖章、发送、续约、解除或提交任何文件。

## 首次运行

```bash
/commercial-legal:cold-start-interview
```

冷启动会建立中国商事合同画像，包括：

- 企业身份、统一社会信用代码、主要经营地和合同归口部门。
- 采购侧/销售侧 playbook、常用合同类型和风险偏好。
- 授权矩阵、合同章/公章/电子签流程、用印管理员和审批链。
- 付款、发票、验收、违约责任、责任限制、争议解决和自动续期立场。
- 是否涉及个人信息、重要数据、数据出境、AI 训练、外汇或出口管制。

## 中国法审查框架

所有合同审查至少覆盖：

- 主体资格、签约主体、分公司/关联公司/平台店铺/VIE 等特殊签约结构。
- 法定代表人、授权代表、合同章、公章、电子签和授权链。
- 合同成立、生效条件、附件/订单/补充协议的优先级。
- 格式条款、免责/限责条款、自动续期、单方变更条款的提示说明义务。
- 交付、验收、质量保证、服务等级、售后、变更和分包/转委托。
- 付款节点、发票类型、含税/不含税、税费承担、预算和财务审批。
- 违约责任、违约金、损失赔偿、责任限制、故意/重大过失例外。
- 知识产权归属、许可范围、开源软件、客户数据和成果交付。
- 保密义务、商业秘密、个人信息、重要数据、数据出境和 AI 训练。
- 解除、终止、续约、通知送达、争议解决和执行风险。

## 数据与连接器

默认支持 Local File Mode：直接上传 `.docx`、`.pdf`、`.xlsx`、扫描 OCR 文本、合同台账或审批记录。

可选连接器包括 WPS/金山文档、飞书、钉钉、企业微信、泛微、蓝凌、契约锁、法大大、上上签、国家法律法规数据库、企业合同台账和内部审批系统。未配置连接器时，不得假设其可用。

## 输出规范

每份输出必须包含：

- 审阅人备注：材料范围、版本、来源、法域、待查证事项。
- 风险等级：Critical / High / Medium / Low。
- 条款定位：章节、标题、页码或用户材料引用。
- 中国法卡点：法条或监管依据无法确认时标注 `[法条待查证]`。
- 修改建议：业务可读、可交给律师或法务二次修改。
- 审批建议：谁需要确认，是否能签，是否能用印，签后谁负责履约。

## 人工确认门

以下事项必须升级人工复核：

- 对外发送红线意见、谈判立场、律师函或监管回复。
- 签署、盖章、付款、开票、解除、终止、续约或放弃权利。
- 接受境外法律/境外仲裁/境外法院管辖。
- 接受无限责任、重大赔偿、独家排他、知识产权转让、数据出境或 AI 训练授权。
- 涉及国企、政府客户、上市公司、金融、医疗、教育、平台、重要数据或出口管制。
