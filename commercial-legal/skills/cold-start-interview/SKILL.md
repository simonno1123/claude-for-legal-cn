# /cold-start-interview

## 强制规则

- 默认法域为中国大陆；不得询问 Delaware C-corp、LLC、state bar、DocuSign/Slack/Google Drive 作为默认生态。
- 若用户要求更新配置，仍通过本技能处理；不得使用独立 `customize` 入口。
- 任何用户提供的法条、阈值、资质、登记号、授权事实都标注来源；无法核验时写 `[待查证]`。

## 目标

生成 `~/.claude/plugins/config/claude-for-legal/commercial-legal/CLAUDE.md`，作为所有商事合同审查技能的唯一画像来源。

## 访谈维度

### 1. 企业和法域

- 公司名称、统一社会信用代码、公司类型、主要经营地。
- 默认合同法域：中国大陆 / 涉港澳台 / 涉外。
- 常见合同身份：采购方、销售方、服务提供方、客户、平台方、代理商。
- 是否有国企、政府客户、上市公司、金融、医疗、教育、平台或出口管制场景。

### 2. 合同类型和资料来源

- 采购合同、销售合同、SaaS、云服务、技术服务、软件许可、NDA、框架协议、订单、补充协议。
- 合同资料来源：本地上传、WPS/金山文档、飞书、钉钉、企业微信、泛微、蓝凌、契约锁、法大大、上上签、合同台账。
- 未配置连接器时，启用 Local File Mode，不假设可自动读取。

### 3. 授权、签署和用印

- 谁能签：法定代表人、授权代表、业务负责人、平台账号。
- 需要什么：授权委托书、OA 审批、合同审批单、董事会/股东会决议。
- 用什么章：合同章、公章、电子签、骑缝章。
- 电子签平台和证据留存方式。
- 用印管理员和用印前置审批。

### 4. Playbook

分别采集采购侧和销售侧：

- 责任上限、违约金、间接损失、故意/重大过失例外。
- 付款节点、发票类型、含税/不含税、税费承担。
- 验收、质量保证、服务等级、数据迁移和退出。
- 知识产权归属、许可、成果交付、开源软件。
- 自动续约、涨价、取消通知和预算审批。
- 适用法律、法院管辖/仲裁机构、仲裁地、语言和送达方式。

### 5. NDA 和数据

- 保密协议是否单向/双向，商业秘密范围，保密期限。
- 是否允许残余知识、反向工程、样品测试、员工/客户不招揽。
- 是否处理个人信息、敏感个人信息、重要数据、客户数据、日志。
- 是否涉及数据出境、境外远程访问、AI 训练、第三方模型。

### 6. 审批矩阵

按金额、合同类型和风险触发器采集：

- 业务负责人、采购、财务/税务、法务负责人、数据合规/信息安全、用印管理员、总经理/董事会。
- 必须升级事项：无限责任、境外争议解决、数据出境、知识产权转让、独家排他、自动续约超阈值、国企/政府客户。

## 禁止事项

- 不得把美国/欧盟模板字段写入默认画像。
- 不得输出“可直接签署”；只能提示进入人工复核或补正流程。
- 不得跳过授权、用印、发票、争议解决和数据合规。
