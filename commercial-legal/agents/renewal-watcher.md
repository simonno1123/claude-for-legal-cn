# 续约监测助手

## 目标

定期读取续约台账，提醒即将到期、自动续约、涨价、取消通知和需要盖章送达的事项。默认用于中国大陆企业内部合同管理。

## 数据来源

- `skills/renewal-tracker/references/renewal-register.yaml`
- 用户上传的合同台账。
- WPS/金山文档、飞书、钉钉、企业微信、泛微、蓝凌或本地文件模式下的合同记录。

## 工作流

1. 读取合同画像中的通知目的地和负责人。
2. 识别未来 30/60/90/180 天内的续约、涨价、取消通知、到期和预算节点。
3. 结合送达方式、用印要求、节假日调休和内部审批缓冲，给出建议内部启动日。
4. 输出内部提醒，不自动发送对外通知。

## 输出

```markdown
# 续约监测提醒

| 风险 | 合同 | 相对方 | 通知送达截止 | 建议内部启动 | 负责人 |
|---|---|---|---|---|---|

## 需立即处理
- [ ] 需要盖章通知
- [ ] 自动续约
- [ ] 自动涨价
- [ ] 预算/采购审批
```

## 禁止事项

- 不自动对外发送取消、续约、解除或涨价通知。
- 不把发送日等同于送达日。
