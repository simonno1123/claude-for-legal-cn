# /clearance

## 定位

这是中国商标清除初筛，不是正式商标可注册性意见。不得输出“可注册”“可使用”“无风险”作为最终结论。正式结论需由中国商标律师或商标代理机构基于 CNIPA 官方检索、近似判断和业务证据作出。

## 强制边界

不得使用境外商标判例法混淆测试作为默认框架。中国版必须适用：

- 《商标法》第10条：绝对禁用条款；
- 《商标法》第11条：缺乏显著性；
- 《商标法》第30条：相同/近似商标与类似商品/服务阻挡；
- 《商标法》第32条：不得损害他人在先权利、不得抢注他人已使用并有一定影响的商标；
- 《商标法》第33条：异议；
- 《商标法》第44条：无效宣告；
- 《商标法》第49条：连续三年不使用撤销；
- CNIPA《商标审查审理指南》；
- 《类似商品和服务区分表》。

## Intake

一次性收集：

| 信息 | 说明 |
|---|---|
| 拟用标志 | 中文、英文、拼音、图形、组合、颜色、含义、读音 |
| 商品/服务 | 实际经营内容，不只写类别 |
| 类别 | 已知尼斯类别；未知时先建议类别并提示需代理机构确认 |
| 使用场景 | 注册、上线、包装、App 名称、店铺名、活动名、域名、Logo |
| 在先检索 | 是否已有中国商标网/CNIPA/代理机构检索结果 |
| 时间计划 | 是否已宣传、是否临近上线、是否已申请 |

## 审查流程

### 1. 绝对条款红线

逐项审查是否触发：

- 国家名称、国旗、国徽、军旗、勋章、中央国家机关名称或标志等；
- 外国国家名称、国际组织名称或标志；
- 带有民族歧视性、欺骗性、不良影响、损害社会主义道德风尚；
- 县级以上行政区划地名或公众知晓的外国地名；
- 其他《商标法》第10条风险。

触发时标为 `Block`，不得建议直接申请。

### 2. 显著性

审查是否仅直接表示商品/服务的通用名称、图形、型号、质量、主要原料、功能、用途、重量、数量或其他特点。缺乏显著性时标为 `Danger` 或 `Needs Review`，并提示可通过改名、图形化、组合化或长期使用取得显著性，但不能保证通过。

### 3. 近似和类似商品/服务

基于“相关公众通常认识”审查：

- 标志近似：字形、读音、含义、图形构图、颜色、整体结构和整体视觉印象；
- 商品/服务类似：严格对照《类似商品和服务区分表》，同时关注实际功能、用途、销售渠道、消费对象；
- 关联类别：App、SaaS、软件服务、硬件、广告、电商、教育、医疗等常见跨类风险；
- 在先权利：字号、姓名、肖像、著作权、域名、有一定影响的商品名称/包装装潢。

### 4. 清除策略

发现阻挡商标时，必须输出可行路径：

- 公告期内：异议；
- 已注册满三年且可能未使用：撤三；
- 恶意注册、违反绝对条款或损害在先权利：无效宣告；
- 类别或商品服务可调整：限缩商品/服务、改名或共存评估；
- 阻挡商标权利人注销、吊销或失联：重点评估撤三成功率和证据缺口。
