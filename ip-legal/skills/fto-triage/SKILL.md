# /fto-triage

## 定位

这是中国专利自由实施分析初筛，不是正式 FTO 法律意见。不得输出“可自由实施”“不侵权”“可上市”作为最终结论。正式意见需由中国专利律师或专利代理师结合全面检索、权利要求解释、稳定性分析和业务证据作出。

## 强制中国法框架

- 《专利法》第11条：未经许可不得实施发明、实用新型、外观设计专利；
- 《专利法》第64条：保护范围以权利要求内容为准，说明书和附图可解释权利要求；
- 《专利法》第65条：专利侵权纠纷处理路径；
- 《专利法》第66条：实用新型和外观设计的专利权评价报告；
- 《专利法》第67条：现有技术/现有设计抗辩；
- 《专利法》第71条：损害赔偿和惩罚性赔偿；
- 最高人民法院专利侵权司法解释中的全面覆盖原则、等同原则和禁止反悔规则。

## Intake

| 信息 | 说明 |
|---|---|
| 技术方案 | 产品结构、工艺流程、软件/硬件功能、关键技术点 |
| 行为类型 | 制造、使用、许诺销售、销售、进口 |
| 地域 | 中国大陆制造/销售/进口/出口/OEM |
| 已知专利 | 专利号、权利人、权利要求、状态 |

## 审查流程

### 1. 专利类型分流

必须区分：

- 发明专利：关注权利要求解释、保护期限、年费、无效风险；
- 实用新型：高风险重点，数量多、不经实质审查授权，必须审查或申请《专利权评价报告》；
- 外观设计：按整体视觉效果和相同/相近种类产品判断，必须审查外观设计评价报告；
- 无效、终止、届满或年费状态不明：标注 `[官方状态待查证]`。

### 2. 检索要求

必须提示覆盖：

- CNIPA 中国专利检索；
- 实用新型检索；
- 外观设计检索；
- 主要竞争对手和关联公司专利；
- 关键词、分类号、申请人、发明人、同族专利。

没有数据库时，明确写：“未运行官方/商业专利数据库检索，本分析仅基于用户提供专利，不能作为上市依据。”

### 3. 权利要求比对

对每个高风险专利的独立权利要求逐项比对：

| 权利要求要素 | 产品是否具备 | 依据 | 风险 |
|---|---|---|---|

规则：

- 全面覆盖原则：产品落入某项权利要求全部技术特征，才构成字面侵权高风险；
- 缺少任一必要技术特征时，字面侵权风险降低，但仍需评估等同；
- 等同原则单独提示，不能自行作最终判断；
- 权利要求解释、禁止反悔和捐献原则需律师/代理师确认。

### 4. 稳定性和抗辩

必须检查：

- 实用新型/外观设计是否已有《专利权评价报告》；
- 是否存在现有技术/现有设计抗辩；
- 是否可提出专利权无效宣告请求；
- 是否有许可、交叉许可、标准必要专利或不侵权抗辩；
- 是否涉及 OEM、出口、海关扣留或行政裁决。
