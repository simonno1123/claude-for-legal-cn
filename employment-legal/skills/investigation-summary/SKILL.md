# /investigation-summary

## 适用场景

当内部调查台账或调查备忘录已经形成，需要向不同受众同步事实、证据缺口、处分/解除风险或后续行动时，使用本技能生成分层摘要。

## 中国法强制声明

不得使用英美法“特权备忘录”“工作成果豁免”“律师心证”“可对抗证据开示”等表述。中国法下只能表述为：

- 内部合规调查摘要；
- 保密审阅初稿；
- 法务专业评估意见/内部研判；
- 仅供内部决策参考，需律师或企业法务复核。

该摘要不具有对抗劳动仲裁、人民法院或行政机关调取的证据特权。

## Instructions

1. 加载 `internal-investigation` 参考技能，并运行 Mode 5（Audience summary）。
2. 如尚无调查备忘录，先提示用户补充调查台账、访谈记录、证据清单或要求先生成调查备忘录。
3. 按受众控制披露范围：
   - `hr`：突出事实、证据缺口、程序动作、员工沟通口径，不展开内部法律争议推演。
   - `management`：突出风险等级、商业影响、是否可处分/解除、预算和升级事项。
   - `outside-counsel`：提供完整事实、证据索引、程序材料、待确认法条和地方口径。
   - `dispute-prep`：聚焦仲裁/诉讼举证责任、证据原件、送达留痕和证人可用性。
4. 所有版本均应删除无关个人信息，避免扩大传播敏感个人信息。

```markdown
# 内部合规调查摘要

> 保密审阅初稿 - 不承诺证据特权
> 受众：[HR / 管理层 / 外部律师 / 仲裁诉讼准备]

## 1. 调查事项

## 2. 已确认事实

## 3. 关键证据与缺口

## 4. 法律与用工风险

## 5. 建议动作
```
