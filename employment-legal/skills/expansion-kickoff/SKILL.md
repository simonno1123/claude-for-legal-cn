# /expansion-kickoff

## Phase 2 挂起声明

涉外用工、EOR、境外实体设立、永久机构风险、境外社保税务和外籍员工签证不属于 `employment-legal` 中国法 Phase 1 的正式能力。第一阶段不得把境外劳动法、美国母公司或 EOR 框架作为默认结论。

## Instructions

1. 告知用户该能力已挂起至 Phase 2。
2. 如用户确需推进涉外用工，仅输出问题清单，不给出具体国家法律结论：
   - 目标国家/地区；
   - 计划岗位、人数、雇佣主体；
   - 是否已有当地实体；
   - 是否涉及中国员工外派、跨境数据、个税、社保、签证；
   - 需要当地律师、税务顾问和移民顾问确认的事项。
3. 建议用户转入独立涉外用工项目或由当地律师出具意见。

## 输出格式

```markdown
# 涉外用工扩张提示

**状态：** Phase 2 暂缓
**本模块不输出：** EOR/实体设立选择、境外解除、境外强制福利、境外税务或移民法律结论
**建议：** 由目标法域当地律师、税务顾问和移民顾问确认
```
