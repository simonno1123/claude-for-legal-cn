# Internal Investigation（中国版共享框架）

## 中国法域与引用规则（强制）

- 默认适用中华人民共和国大陆地区劳动法律体系。
- 不得使用 Upjohn warning、Weingarten、Garrity、attorney work product、discovery privilege 等美国调查法或证据特权概念。
- 调查材料标注为“内部合规调查纪要 / 严密保密级文件 / 律师或法务审阅初稿”，不得承诺在中国仲裁或诉讼中享有证据披露豁免特权。
- 涉及解除、处分、报案、移送监管、对外函件前，必须设置法务/律师人工确认门。

## 调查目标

本框架服务于中国企业内部调查，尤其是配合《劳动合同法》第39条过错解除的证据链建设。重点判断：

- 是否存在客观违纪事实。
- 违纪事实是否能被证据证明。
- 员工是否有陈述申辩机会。
- 员工手册或制度是否满足内容合法、民主程序、公示告知。
- 处分或解除是否完成内部审批、工会通知和送达留痕。

## 调查材料类型

优先收集：

- 员工手册、奖惩制度、反舞弊制度、信息安全制度、费用报销制度等制度文本及生效证据。
- 监控录像、门禁记录、考勤记录、系统日志、邮箱/企业微信/钉钉/飞书记录。
- 报销发票、付款记录、采购单据、合同、审批流、财务凭证。
- 员工访谈笔录、陈述与申辩书、情况说明、承诺书。
- 证人访谈纪要、部门负责人说明、HR 沟通记录。
- 送达留痕：微信、邮件、EMS、快递面单备注、OA 已读回执。

## Mode 1：开启调查

用于 `/employment-legal:investigation-open`。

询问：

- 调查事项名称和触发原因。
- 涉及员工、岗位、部门、工作地。
- 违纪类型：旷工、舞弊、报销违规、利益冲突、泄密、性骚扰、暴力冲突、信息安全、失职造成损失等。
- 拟依据的员工手册或制度章节。
- 是否可能走第39条解除、处分、调岗、降薪、报案或索赔。
- 是否已建立工会。
- 是否已有初步证据和员工申辩材料。

建立调查台账字段：

```yaml
matter: "[调查事项]"
opened: "[日期]"
employee: "[姓名/编号]"
department: "[部门]"
work_location: "[城市]"
allegation: "[违纪事实摘要]"
possible_action: "[处分/解除/报案/索赔/待定]"
handbook_basis: "[制度章节]"
union_notice_needed: true
status: open
entries: []
evidence_gaps: []
```

## Mode 2：添加材料

用于 `/employment-legal:investigation-add`。

每条材料必须记录：

- 来源：员工、证人、系统、监控、财务、HR、法务、第三方。
- 取得方式和日期。
- 是否为原件、截图、导出件、复印件或转述。
- 是否能证明违纪事实、严重性、制度依据、员工知悉、损失后果或送达。
- 是否需要补强，例如原始日志、盖章说明、签字确认、平台导出证明。

证据分级：

| 等级 | 说明 |
|---|---|
| A | 原始记录或可核验系统导出，来源清晰，时间完整 |
| B | 截图、复印件、转发记录，有一定佐证但需补强 |
| C | 单方陈述或间接材料，需要其他证据印证 |
| Gap | 应存在但尚未取得的关键证据 |

访谈材料要求：

- 访谈笔录应载明时间、地点、访谈人、被访谈人、在场人员。
- 关键事实应尽量问答式记录。
- 员工本人的陈述与申辩书或访谈笔录应逐页签字，重要页建议捺印。
- 员工拒签时，应记录拒签过程，并由见证人签字或保留录音录像/送达留痕。

## Mode 3：查询调查台账

用于 `/employment-legal:investigation-query`。

回答时按问题类型输出：

- 事实查询：引用台账条目编号和证据等级。
- 冲突查询：列明员工陈述、证人陈述和客观证据之间的矛盾。
- 缺口查询：列出影响处分或解除稳定性的证据缺口。
- 严重性查询：说明是否足以支撑“严重违反规章制度”或“严重失职、营私舞弊给用人单位造成重大损害”。
- 程序查询：检查制度三要件、陈述申辩、工会通知、送达留痕。

## Mode 5：生成不同受众摘要

用于 `/employment-legal:investigation-summary`。

摘要必须按受众降噪：

- HR 摘要：事实、证据缺口、下一步流程，不给最终解除决定。
- 管理层摘要：风险等级、业务影响、需决策事项。
- 外部律师摘要：完整事实、制度依据、证据缺口、拟处分路径和待问问题。
- 员工沟通稿：仅在法务确认后生成，避免承认未查清事实或作出过度定性。

## 解除/处分人工确认门

以下情形必须升级：

- 拟依据第39条解除。
- 证据主要来自截图、单方陈述或无法核验材料。
- 员工拒绝签字、拒绝访谈或提出反向投诉。
- 员工处于孕期、产期、哺乳期、医疗期、工伤停工留薪期。
- 员工涉及工会干部、举报人、职业病、伤残或其他特殊身份。
- 可能报案、索赔、对外披露或移送监管。

## 禁止事项

- 不得承诺调查纪要享有美国式证据特权。
- 不得把“怀疑”写成“已查明”。
- 不得用制度条款替代事实证据。
- 不得忽略员工陈述申辩。
- 不得在证据缺口未补齐时建议直接解除。
