# /severance-calculator

## 适用场景

当用户要求“算 N / N+1 / 2N”“解除补偿多少”“协商解除底线”“违法解除赔偿金”“三倍封顶怎么套”时使用本技能。

本技能只输出测算口径和风险提示，不替代工资审计、地方口径核验或律师/法务最终确认。

## 必要输入

| 参数 | 说明 |
|---|---|
| 入职日期 | 用于计算工作年限 |
| 拟解除/终止日期 | 用于计算工作年限和平均工资期间 |
| 解除路径 | 第39条、第40条、第41条、协商解除、合同到期终止、违法解除风险等 |
| 离职前12个月平均工资 | 应说明是否包含奖金、提成、津贴、补贴、加班费等 |
| 工作地点/社保缴纳地 | 用于查证本地区上年度职工月平均工资 |
| 本地区上年度职工月平均工资 | 未提供时标注 `[地方规定 — 待查证]` |

缺少任一关键参数时，不得给出确定金额，只能输出“待补参数测算表”。

## 强制法条映射

- 是否应支付 N，先按《劳动合同法》第46条判断触发事由，再按第47条测算。
- 劳动者依据第38条被迫解除并主张经济补偿时，必须提示地方裁审口径差异，尤其社保少缴、异地代缴、公积金争议需标注 `[地方规定 - 待查证]`。
- 违法解除/终止后果先看第48条继续履行；不能继续履行或劳动者不要求继续履行的，按第87条以第47条经济补偿标准的二倍计算赔偿金。
- 第42条保护期人员不得通过第40条、第41条路径以 N 或 N+1 强行解除。

### 1. N（经济补偿）

```text
经济补偿（N） = 补偿年限 × 离职前12个月平均工资
```

年限折算：

- 每满一年支付一个月工资。
- 六个月以上不满一年的，按一年计算。
- 不满六个月的，支付半个月工资。

### 2. N+1

仅在第40条路径下，企业未提前30日书面通知时，除 N 外另付一个月工资。不得把所有协商解除或第39条解除都泛化为 N+1。

代通知金通常按劳动者本人一个月工资口径提示，不得当然套用第47条的三倍社平工资和12年封顶规则；若地方口径不同，标注 `[地方规定 — 待查证]`。

### 3. 2N

违法解除赔偿金通常按经济补偿标准的二倍计算。不得叠加表述为“2N+N”。如劳动者请求继续履行劳动合同，需提示恢复劳动关系风险。

### 4. 三倍封顶

如劳动者月工资高于用人单位所在直辖市、设区的市级人民政府公布的本地区上年度职工月平均工资三倍：

- 月工资按三倍社平工资封顶。
- 支付经济补偿的年限最高不超过十二年。
- 社平工资年度、地区、发布主体和适用口径必须标注 `[地方规定 — 待查证]`。

## 测算流程

1. 确认解除路径：第39条、第40条、第41条、协商解除、合同期满终止、违法解除风险。
2. 判断是否适用 N、N+1、2N 或无经济补偿。
3. 计算工作年限折算。
4. 核验离职前12个月平均工资口径。
5. 判断是否触发三倍封顶。
6. 输出测算表、缺口和复核事项。

## 禁止事项

- 不得说“支付 severance 即可解除”。
- 不得在缺少工资基数、工作年限或地方社平工资时给出确定金额。
- 不得把第39条零补偿、第40条 N+1、违法解除 2N 混为一个通用离职金概念。
