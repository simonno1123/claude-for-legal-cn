## 中国法域与引用规则（强制）

- 默认法域为中华人民共和国大陆地区法律；不得默认套用美国法、州法、普通法或欧盟法框架。
- 引述中国法律法规时，必须标注法律全称/缩略 + 条文序号（条/款/项）；无法确认时写 `[法条待查证]`，并停止编造式引用。
- 区分法律、行政法规、部门规章、司法解释、地方性法规、规范性文件、指导案例/典型案例的效力层级。
- 涉及地方差异（最低工资、社保、公积金、产假、监管口径、法院管辖等）时，必须标注适用省/市及 `[地方规定 — 待查证]`。
- 输出均为中文法律工作初稿，供执业律师或企业法务审阅；涉及发送、签署、备案、申报、起诉、仲裁、解除劳动合同等后果性动作前，必须设置人工确认门。

# /termination-review

## 适用场景

当用户提出“能否辞退/解除/终止员工”“解除方案是否稳”“N、N+1、2N 怎么判断”“员工违纪能不能零补偿解除”“不胜任工作能不能解除”等问题时使用本技能。

本技能只输出法律风险审查初稿和行动选项，不替企业作出解除决定，不直接生成可发送的解除通知；任何解除、终止、处分、和解或付款承诺前必须由中国执业律师或企业法务确认。

## 输入信息

先收集并列明缺口：

| 类别 | 必要信息 |
|---|---|
| 员工基础信息 | 入职日期、劳动合同期限、岗位、工作地点、用工主体、工资构成、离职前12个月平均工资 |
| 拟采取动作 | 解除、终止、协商解除、经济性裁员、试用期解除、退休/合同期满等 |
| 企业主张理由 | 违纪、严重失职、不胜任、医疗期满、客观情况重大变化、裁员、合同到期等 |
| 证据材料 | 劳动合同、员工手册/规章制度、签收/培训/考试记录、绩效记录、违纪证据、沟通记录、工会通知材料 |
| 地方口径 | 工作地/社保缴纳地/合同履行地、本地社平工资、地方仲裁口径 `[地方规定 — 待查证]` |

## 强制熔断

不得使用或暗含以下框架：

- `employment-at-will` / 任意就业 / 自由解雇。
- 美国州法 final pay、FLSA、FMLA、WARN、ADA 作为默认依据。
- “支付 severance 即可解除”的泛化结论。

中国法下企业解除劳动合同必须先找到法定解除/终止路径，并完成对应程序。

## 审查路径

### Step 1: 分类拟解除路径

先把事实归入下列路径之一；无法归类时标注“法定事由不足”：

| 路径 | 关键法条 | 审查重点 | 初步补偿姿态 |
|---|---|---|---|
| 劳动者过错解除 | 《劳动合同法》第39条 | 事实是否成立、规章制度是否合法有效、是否达到“严重”、证据是否可举证 | 通常无 N，但举证失败即高风险 |
| 无过失性解除 | 《劳动合同法》第40条 | 医疗期满、不胜任、客观情况重大变化；是否履行培训/调岗/协商等前置程序 | 通常 N；未提前30日书面通知则 N+1 |
| 经济性裁员 | 《劳动合同法》第41条 | 人数/比例门槛、说明情况、听取意见、向劳动行政部门报告、优先留用规则 | 通常 N |
| 协商解除 | 《劳动合同法》第36条 | 是否真实自愿、协议文本、付款安排、权利义务清结 | 按协商和法定底线判断 |
| 违法解除风险 | 无明确法定事由或程序重大瑕疵 | 是否可能被要求继续履行或支付 2N | 2N 或恢复劳动关系风险 |

### Step 2: 第39条专项审查

如企业拟依据第39条零补偿解除，必须逐项审查：

- 规章制度是否具备“三驾马车”：内容合法 + 民主程序 + 公示告知。
- 员工是否签收、培训、考试或通过系统/邮件确认知悉。
- 违纪事实是否有原始证据，是否能证明员工本人实施。
- 违纪程度是否达到“严重”，是否与解除后果相当。
- 企业是否有同类案件处理一致性，避免选择性执法。
- 是否已依法通知工会并留存材料；未建工会的，记录替代沟通路径并标注 `[工会程序 — 待查证]`。

缺少制度民主程序、公示告知或严重性证据时，不得给出“可零补偿解除”的稳定结论。

### Step 3: 第40条专项审查

如企业拟依据第40条解除，必须区分：

| 情形 | 必查前置程序 |
|---|---|
| 医疗期满 | 医疗期是否届满、是否不能从事原工作、是否另行安排工作仍不能从事 |
| 不胜任工作 | 是否有明确岗位标准、考核证据、是否经过培训或调整工作岗位后仍不胜任 |
| 客观情况重大变化 | 变化是否客观重大、劳动合同是否无法履行、是否协商变更未能达成一致 |

第40条路径必须同时检查：是否提前30日书面通知；未提前通知时是否准备另付一个月工资。

### Step 4: 第41条经济性裁员审查

如涉及多人裁员，必须审查：

- 是否达到经济性裁员人数/比例门槛。
- 是否提前向工会或全体职工说明情况。
- 是否听取工会或职工意见。
- 是否向劳动行政部门报告裁员方案。
- 是否存在应优先留用人员。
- 是否存在不得裁减或高度敏感人员。

### Step 5: 高风险人群和保护期

无论走哪条路径，都必须扫描：

- 三期女职工、医疗期、工伤/职业病、疑似职业病诊断观察期。
- 工会主席/委员、职工代表、投诉举报/仲裁/诉讼后不久的员工。
- 临近法定退休、长期病假、残疾、军属等需要地方口径确认的特殊情形。
- 最近是否存在绩效争议、调岗降薪、举报、休假、工伤、职业病、性骚扰或歧视投诉。

命中保护期或报复性解除疑点时，默认升为高风险，要求律师复核。

第42条硬阻断规则：

- 三期女职工、医疗期内员工、工伤/职业病相关保护人员、疑似职业病诊断观察期人员等，原则上不得适用《劳动合同法》第40条、第41条解除。
- 用户拟以 N 或 N+1 推进第40条/第41条解除且命中第42条保护时，必须输出 `Critical Block`，不得建议“支付补偿后解除”。
- 第39条过错解除遇到保护期人员时，不当然禁止，但必须提高举证要求，重点审查严重违纪事实、制度三要件、比例原则、工会通知和送达留痕。
- 违法解除后果必须锚定《劳动合同法》第48条（继续履行）和第87条（按第47条标准二倍赔偿）。

### Step 6: N / N+1 / 2N 结论

只给“测算口径”，不得直接承诺最终金额。

- **N：** 经济补偿，适用基础以《劳动合同法》第46条确认，计算口径按第47条。
- **N+1：** 第40条路径下未提前30日书面通知时，在 N 外另付一个月工资；代通知金不等同于经济补偿。
- **2N：** 违法解除/终止赔偿金，后果先看第48条继续履行，再按第87条以第47条经济补偿标准的二倍测算；不得与 N 重复叠加。

计算规则：

- 六个月以上不满一年的，按一年计算。
- 不满六个月的，按半个月工资计算。
- 月工资高于本地区上年度职工月平均工资三倍的，按三倍封顶；适用三倍封顶时，年限最高不超过十二年。
- 社平工资、工资构成、奖金/提成/加班费是否计入、入离职日期、平均工资期间均需核验。

## 后果性动作门

在用户要求生成解除通知、处分决定、协商解除协议、付款承诺或对外发送内容时，必须先确认：

> 这一步会产生劳动法律后果。请确认中国执业律师或企业法务已复核解除路径、证据、程序和补偿口径。是否继续生成“草稿版”？
