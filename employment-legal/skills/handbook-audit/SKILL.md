## 中国法域与引用规则（强制）

- 默认法域为中华人民共和国大陆地区法律；不得默认套用美国法、州法、普通法或欧盟法框架。
- 引述中国法律法规时，必须标注法律全称/缩略 + 条文序号（条/款/项）；无法确认时写 `[法条待查证]`，并停止编造式引用。
- 区分法律、行政法规、部门规章、司法解释、地方性法规、规范性文件、指导案例/典型案例的效力层级。
- 涉及地方差异（最低工资、社保、公积金、产假、监管口径、法院管辖等）时，必须标注适用省/市及 `[地方规定 — 待查证]`。
- 输出均为中文法律工作初稿，供执业律师或企业法务审阅；涉及发送、签署、备案、申报、起诉、仲裁、解除劳动合同等后果性动作前，必须设置人工确认门。

# /handbook-audit

## 目标

对中国企业员工手册和规章制度做全量体检，判断其是否可作为劳动仲裁/诉讼中处分、解除、调岗、考核、扣减奖金等管理动作的稳定依据。

本技能的核心不是“润色制度文本”，而是审查规章制度的可执行性和证据链。

## 必要材料

| 材料 | 用途 |
|---|---|
| 员工手册/规章制度全文 | 审查内容合法性和条款清晰度 |
| 职工代表大会或全体职工讨论材料 | 审查民主程序 |
| 工会或职工代表协商材料 | 审查平等协商过程 |
| 发布/公示/培训/考试/签收记录 | 审查公示告知 |
| 历史处分或解除案例 | 检查一致性和选择性执法风险 |
| 工作地点/用工地区 | 检查地方工资、休假、社保、公积金口径 |

缺少民主程序或公示告知材料时，必须把“可作为解除依据”评为高风险或待补证。

## 三驾马车审查

### 1. 内容合法

逐章检查：

- 招聘录用、试用期、岗位调整、绩效管理。
- 工时、考勤、加班、休息休假。
- 薪酬、奖金、提成、扣款、赔偿。
- 保密、竞业限制、知识产权。
- 奖惩、违纪、严重违纪、解除。
- 数据、监控、个人信息处理。
- 反骚扰、反歧视、安全生产、职业健康。
- 离职交接、费用报销、财物归还。

重点标出：

- 违反法律强制性规定的条款。
- 用语过宽、严重程度不清、可任意解释的条款。
- 罚款、扣工资、降薪、调岗、解除等高风险条款。
- 与劳动合同、offer、集体合同、地方规定冲突的条款。

### 2. 民主程序

涉及劳动者切身利益的规章制度，必须检查是否有：

- 职工代表大会或全体职工讨论记录。
- 方案和意见征集记录。
- 工会或职工代表平等协商记录。
- 最终版本与讨论版本的一致性说明。

没有民主程序材料时，不得输出“制度有效无重大问题”。应标注：

> 民主程序缺口：该制度作为处分/解除依据存在被仲裁委或法院否认的风险。

### 3. 公示告知

检查是否有：

- 员工签收。
- 系统公告阅读确认。
- 邮件送达和回执。
- 培训签到、考试记录。
- 新员工入职制度确认。
- 修订版本的再次告知记录。

没有公示告知留痕时，不得建议企业直接依据该制度解除劳动合同。

## 高风险条款清单

必须单独输出以下条款风险：

| 条款类型 | 典型风险 |
|---|---|
| 严重违纪 | “严重”标准不清、缺少阶梯处分、证据要求不足 |
| 旷工解除 | 天数标准、通知程序、考勤证据、特殊情况说明 |
| 绩效淘汰 | 末位淘汰、强制调岗降薪、未设置改进期 |
| 薪酬扣减 | 罚款、赔偿、奖金取消、提成不发 |
| 加班审批 | 未审批不支付加班费的绝对化表述 |
| 竞业限制 | 人员范围过宽、补偿标准缺失、期限/地域/范围过宽 |
| 个人信息与监控 | 未告知处理目的、范围、保存期限和员工权利 |
| 调岗调薪 | 单方调整权限过宽，缺少合理性和协商机制 |

## 后果性动作门

如用户要求“据此辞退/处分/降薪/调岗”，必须先提示：

> 该动作可能产生劳动争议后果。请确认律师/法务已复核制度内容、民主程序、公示告知和具体事实证据。是否仅生成“内部讨论草稿”？
