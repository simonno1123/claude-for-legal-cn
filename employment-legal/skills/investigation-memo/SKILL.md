# /investigation-memo

## 中国法域与引用规则（强制）

- 默认适用中华人民共和国大陆地区劳动法律体系。
- 不得使用 `ATTORNEY WORK PRODUCT`、Upjohn、discovery privilege 等美国法标签。
- 输出标题使用“内部合规调查纪要 / 严密保密级文件 / 律师或法务审阅初稿”。
- 不得承诺该文件在中国仲裁、诉讼或监管调查中享有英美法证据披露豁免特权。

## 执行方式

1. 读取 `internal-investigation` 共享框架。
2. 运行 Mode 4：生成或更新调查纪要。
3. 若调查台账不存在，先提示用户运行 `/employment-legal:investigation-open`。
4. 若新增材料尚未入账，先提示运行 `/employment-legal:investigation-add`。
5. 生成纪要前必须检查证据缺口，不得把未查清事实写成已查明事实。

## 必查卡点

生成纪要前逐项检查：

- 是否存在客观证据支持违纪事实：监控、系统日志、考勤、审批流、报销发票、聊天记录、邮件、门禁、财务凭证等。
- 员工是否提交《陈述与申辩书》或形成访谈笔录。
- 员工访谈笔录是否载明时间、地点、访谈人、被访谈人、在场人员。
- 员工本人是否逐页签字；关键页是否建议捺印。
- 拒签时是否有见证、录音录像、邮件/微信/EMS 送达等留痕。
- 员工手册或制度是否满足内容合法、民主程序、公示告知。
- 拟依据第39条解除时，是否已经或拟通知工会。
- 是否存在三期、医疗期、工伤停工留薪期、职业病、工会干部、举报人等特殊保护。

## 人工确认门

以下情形不得直接生成最终处分或解除建议，必须升级法务/外部律师：

- 证据缺口影响事实认定。
- 员工拒签、拒访或提出反向投诉。
- 拟解除员工属于三期、医疗期、工伤、职业病、工会干部、举报人等特殊群体。
- 公司拟报案、索赔、对外披露或向监管提交材料。
- 员工手册民主程序或公示告知证据不足。
